<html> и </html> - внутри содержится веб-страница
<head> </head> — внутри этих тегов размещают «голову»
<body> </body> — этот элемент предназначен для хранения содержания веб-страницы (контента)
Для заголовков используются теги <h1> </h1>, <h2> </h2> и так далее до <h6> </h6> (чем меньше цифра, тем «важнее» заголовок)
Обычный текст должен включаться в теги <p> </p> — это теги для выделения параграфов
<a href="ссылка">текст ссылки</a> — поддерживает возможность открытия документа
<form> </form> — форма, которую заполняет пользователь
<footer> </footer>	«подвал» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация
Нумерованный список <ol>.
Маркированный список <ul>.
*Каждый пункт списка помещается в парный тег <li>


<br>	перевод строки на новую строку.
<hr>	горизонтальная линия.
<img>	вставляет изображение (фотография, картинка, рисунок и так далее).
<input>	элемент формы, может быть кнопкой, текстовым полем, переключателем и так далее.
<link>	загружает внешние файлы .css, .xml, .ico.
<meta>	мета-теги.
href — добавление ссылки, например, на внешний ресурс или на другую страницу сайта, обязательный атрибут для ссылок


align	right, left, center	Горизонтальное выравнивание тегов.
valign	 top, middle, bottom	Вертикальное выравнивание тегов внутри HTML-элемента.
background	URL	Расположение фонового изображения.
lang	Принимаемые значения: код языка, н-р, ru, en	Определяет код языка содержимого (контента) в элементе.
width	Числовое значение	Определяет ширину таблиц, изображений или ячеек таблицы.
height	Числовое значение	Определяет высоту таблиц, изображений или ячеек таблицы.

color	h1 { color: blue; }	Цвет текста
font-family	h1 { font-family: verdana; }	Шрифт текста
font-size	h1 { font-size: 300%; }	Размер текста
border	p { border: 1px solid powderblue; }	Границы
padding	p { padding: 30px; }	Пространство внутри границы, отступы
margin	p { margin: 50px; }	Пространство за пределами границы
<meta> — универсальный элемент, который может содержать настройки в атрибутах, meta charset указывает кодировку страницы
